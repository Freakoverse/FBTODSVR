FBT Order Doc: SlimeVR

If you're selling Full Body Trackers (SlimeVR) and you'd like to have a simple tool to help you get information from buyers, then this one should do just that.

Just send it to someone that wants to buy trackers from you, have them fill up the fields below, and then they'd be able to download a pdf copy that they can send you.

In regards to the pricing, the price per tracker is set with a default value of $35, and a tracker with an extension tracker at a default value of $45. Also, for each tracker (and extension) comes with a free strap, however, if buyer has more straps than trackers/extensions, then it'll calculate an extra $10).

You can change these values in the 'pdfCreateDownload.js' file when you have this version of this site up on your own domain/gitPage (Thanks for the credit! =3).

The download pdf function works on pc via a browser's print function. If it doesn't work on mobile, then just take a screenshot and send it to the seller.
